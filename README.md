1. What is the difference between var, let, and const?
var: Function-scoped, can be redeclared and reassigned.
let: Block-scoped, can be reassigned but cannot be redeclared in the same scope.
const: Block-scoped, cannot be reassigned or redeclared after initialization.
2. What is the difference between map(), forEach(), and filter()?
map(): Creates and returns a new array by transforming each element.
forEach(): Executes a function for each element but does not return a new array.
filter(): Creates and returns a new array containing only the elements that satisfy a condition.
3. What are arrow functions in ES6?

Arrow functions are a shorter syntax for writing functions in ES6. They use the => operator and have a lexical this, meaning they inherit this from their surrounding scope.

4. How does destructuring assignment work in ES6?

Destructuring allows you to extract values from arrays or properties from objects and assign them to variables in a concise way.

5. Explain template literals in ES6. How are they different from string concatenation?

Template literals use backticks (`) and ${} to embed variables or expressions directly into strings. Unlike string concatenation with the + operator, they are more readable and support multi-line strings.
